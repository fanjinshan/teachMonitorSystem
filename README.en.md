# 教室监控管理系统

## 项目简介

本项目是一套基于 Qt 的局域网教室监控系统，包含教师端和学生端。教师可以实时查看学生的在线状态、违规应用、屏幕截图和摄像头画面，同时支持文件共享、班级管理等教学辅助功能。学生端自动检测前台应用（黑名单），一旦发现违规行为立即截图上报，并支持定期巡查和教师主动请求屏幕/摄像头。

---

## 功能特性

### 教师端
- **实时监控表格**：展示学生 IP、姓名、班级、在线状态、当前违规应用、违规次数。
- **筛选与统计**：支持按班级、在线状态筛选，顶部统计当前显示人数及违规警告数。
- **远程查看**：
  - 查看屏幕：实时请求学生端屏幕截图（JPEG 压缩）。
  - 查看摄像头：实时接收学生端摄像头视频流（JPEG 帧）。
- **文件共享**：教师设置共享文件夹，学生端可浏览并下载文件，自动记录下载次数和来源 IP。
- **班级管理**：创建/删除班级，学生端可加入班级，教师端按班级分组管理。
- **违规日志**：自动保存违规截图和记录到 SQLite 数据库，支持历史查询。

### 学生端
- **前台应用监控**：每秒检测当前活动窗口的标题和进程名，匹配黑名单（短视频、游戏、社交等），白名单（IDE、会议软件等）自动放行。
- **违规上报**：检测到违规立即截屏并通过 TCP 上报教师端；恢复正常后发送恢复信号（10 秒冷却期，避免重复）。
- **定期巡查**：每 5 分钟自动截屏上报，用于课堂巡查。
- **实时响应**：接收教师端的屏幕/摄像头请求，即时截屏或推送摄像头流。
- **文件下载**：浏览教师共享文件夹，下载文件到本地指定路径。
- **个人设置**：修改昵称、头像、选择班级。

## 技术栈

| 组件 | 技术 |
|------|------|
| 开发框架 | Qt 5.15.2 (MinGW 64-bit) |
| 网络通信 | UDP（发现、心跳、命令）、TCP（文件传输、监控数据、视频流） |
| 数据库 | SQLite（学生信息、违规日志、文件统计） |
| 多媒体 | Qt Multimedia（摄像头采集、JPEG 编码） |
| 平台 | Windows（前台应用检测使用 Win32 API） |
